# 学习路线

对于新手小白要如何学习 C++ 呢？这里总结了一些学习资源。
## 学习书籍

按顺序看这几本书《C++ Primer 第5版》-> 《Effective C++》 ->《Effective stl》 -> 《STL源码剖析》-> 《深度探索C++对象模型》（选看）
